# Quickstart

This is the shortest practical script that creates a dataset, compiles a DSL expression, and runs it.

```python
from datetime import UTC, datetime, timedelta
from decimal import Decimal

from laakhay.ta import dataset_from_bars
from laakhay.ta.expr.dsl import compile_expression

start = datetime(2026, 1, 1, tzinfo=UTC)
bars = []
price = Decimal("100")
for i in range(120):
    ts = start + timedelta(hours=i)
    drift = Decimal("0.1") if i % 7 else Decimal("-0.3")
    open_ = price
    close = open_ + drift
    high = max(open_, close) + Decimal("0.2")
    low = min(open_, close) - Decimal("0.2")
    volume = Decimal("1000") + Decimal(i)
    bars.append(
        {
            "timestamp": ts,
            "open": open_,
            "high": high,
            "low": low,
            "close": close,
            "volume": volume,
            "is_closed": True,
        }
    )
    price = close

dataset = dataset_from_bars(bars, symbol="BTCUSDT", timeframe="1h")

expr = compile_expression(
    "close > sma(20) and rsi(14) > 50 and crossup(close, ema(9))"
)

result = expr.run(dataset)
```

## What You Get

- `result` is a dict keyed by `(symbol, timeframe, source)`.
- For a single-symbol OHLCV dataset, key is usually `("BTCUSDT", "1h", "default")`.
- Value is a `Series[bool]` for boolean strategy expressions.
