# Getting Started

This section takes you from install to production-safe expression evaluation.

## Prerequisites

- Python 3.11+ (3.12 recommended).
- Familiarity with OHLCV bars and basic technical indicators.

## Learning Path

1. Install and verify package import.
2. Run a first end-to-end expression over bars.
3. Understand trim/lookback and runtime checks.
4. Move into multi-source patterns.

## Minimal Checklist

- Can construct `Dataset` from bars.
- Can compile and run DSL expression text.
- Can inspect validation/analysis output before execution.
- Can read indicator catalog and choose correct single-output wrappers.
