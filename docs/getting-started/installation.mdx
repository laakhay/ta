# Installation

## Requirements

- Python `>=3.11`
- `pip` or `uv`

## Install From Source

```bash
pip install -e .
```

Or with `uv`:

```bash
uv sync
```

## Verify

```bash
python -c "import laakhay.ta as ta; print('ok')"
```

## Optional Dev Dependencies

```bash
uv run --with pytest python -m pytest -q
```

## Common Setup Issues

- Import errors for `laakhay.ta`: verify you installed from repository root containing the `ta` package.
- Missing indicators in registry after test isolation: re-import `laakhay.ta.indicators` (registry is rebuilt on import).
