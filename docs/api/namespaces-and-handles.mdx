# Namespaces & Handles

## `IndicatorHandle`

`IndicatorHandle` represents an indicator + params and supports algebraic composition.

```python
from laakhay.ta.api import indicator

fast = indicator("sma", period=20)
slow = indicator("sma", period=50)
expr = fast > slow
```

## Namespace Helpers

From `laakhay.ta.api.namespace` (also reachable via `ta` namespace):

- `indicator(name, **params)`
- `literal(value)`
- `source(field)`
- `ref(dataset, timeframe=..., field=..., reference=...)`
- `resample(dataset, from_timeframe=..., to_timeframe=...)`

## `TASeries`

Adapter for method-style indicator calls on a base series.

```python
series_api = ta.ta(close_series)
expr = series_api.sma(period=20) > series_api.ema(period=50)
```

## When To Use Handles

Use handles when building expression graphs programmatically rather than from DSL text.
