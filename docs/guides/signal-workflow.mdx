# Signal Workflow

A robust workflow for strategy signals in this library:

1. Write expression text.
2. Validate and analyze requirements.
3. Preview on historical data.
4. Integrate evaluator/stream path.
5. Add tests for expected trigger behavior.

## Reference Flow

```python
from laakhay.ta.expr.runtime import validate, analyze, preview
from laakhay.ta.expr.dsl import compile_expression

expr_text = "close > sma(50) and rsi(14) > 55"

v = validate(expr_text, exchange="binance")
if not v.valid:
    raise ValueError(v.errors)

a = analyze(expr_text, exchange="binance")

compiled = compile_expression(expr_text)

```

## Interpretation Tips

- Use `preview.triggers` for fast event sanity checks.
- Compare `analyze.min_lookback` with bars loaded into dataset.
- If output is unexpectedly sparse, inspect indicator availability masks and warmup windows.
