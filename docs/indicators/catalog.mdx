## Usage Notes

- Prefer single-output wrappers in DSL roots (for example `stoch_k`, `bb_upper`, `fib_level_down`).
- Multi-output indicators are still valuable in Python-level usage and catalog metadata.
- For Fibonacci strategy logic, use `fib_level_down/up` and `fib_anchor_high/low` with leg and pairing controls.
