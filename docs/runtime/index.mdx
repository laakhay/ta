# Runtime

Runtime tools help you safely execute expressions in research and production workflows.

## Main Components

- `Engine`: minimal node evaluator.
- `Evaluator`: graph evaluator with plan-aware caching and dataset fan-out.
- `RuntimeEvaluator`: dataset-integrated runtime evaluator.
- `preview(...)`: parse/compile/run helper with triggers and indicator extraction.
- `validate(...)`: fast static validation.
- `analyze(...)`: comprehensive requirements/capability analysis.
- `Stream`: incremental update helper with availability transitions.

## Typical Pre-Execution Pipeline

1. `validate(expression)`
2. `analyze(expression, exchange=...)`
3. `preview(expression, dataset=...)`
4. production `compiled_expr.run(dataset)` or stream mode
