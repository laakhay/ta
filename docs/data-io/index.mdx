# Data IO

`laakhay.ta.data` provides pragmatic ingestion helpers for research and service endpoints.

## Main Helpers

- `from_csv(...)` / `to_csv(...)`
- `dataset_from_bars(...)`
- `dataset_from_multisource(...)`
- `trim_dataset(...)`

## Philosophy

- Accept practical payload shapes from APIs.
- Normalize to core immutable types early.
- Keep source labels explicit for deterministic expression resolution.

See:

- `csv` for file-oriented workflows.
- `multisource-datasets` for API payload ingestion.
