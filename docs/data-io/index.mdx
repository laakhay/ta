# Data IO

`laakhay.ta.data` provides pragmatic ingestion helpers for research and service endpoints.

## Main Helpers

Available via `from laakhay.ta import dataset, dataset_from_bars, trim_dataset, from_csv, to_csv`:

- `from_csv(...)` / `to_csv(...)`
- `dataset(...)` – create Dataset from CSV or other sources
- `dataset_from_bars(...)` – from raw OHLCV bar mappings
- `dataset_from_multisource(...)` – from bars, trades, orderbook, liquidations (in `laakhay.ta.data.dataset`)
- `trim_dataset(...)` – trim for indicator lookback

## Philosophy

- Accept practical payload shapes from APIs.
- Normalize to core immutable types early.
- Keep source labels explicit for deterministic expression resolution.

See:

- `csv` for file-oriented workflows.
- `multisource-datasets` for API payload ingestion.
