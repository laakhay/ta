# CSV IO

Use CSV helpers when bootstrapping from flat files.

## Read CSV

```python
from laakhay.ta.data import from_csv

ohlcv = from_csv(
    "./btc_1h.csv",
    symbol="BTCUSDT",
    timeframe="1h",
    timestamp_col="timestamp",
    open_col="open",
    high_col="high",
    low_col="low",
    close_col="close",
    volume_col="volume",
)
```

If OHLCV columns are missing but `value_col` exists, loader returns a generic `Series`.

## Write CSV

```python
from laakhay.ta.data import to_csv

to_csv(ohlcv, "./out/btc_1h.csv")
```

## Mapping Tips

- Always map the timestamp column explicitly when source files vary.
- Keep consistent decimal precision at upstream ETL where possible.
- Sort and deduplicate timestamps before writing if data is merged from multiple vendors.
